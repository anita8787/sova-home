# 🎨 SOVA 品牌配色系統

現代北歐風格，融合焦糖、米白與天然材質色調，輔以點綴色營造「生活儀式感從沙發開始」的品牌氛圍。

---

## 🛠 使用建議

- **Logo 與按鈕主色**：焦糖淺棕 `#B57E4F`
- **背景基底**：米白 `#E8DFC9` + 奶油燕麥 `#FFF8EE`
- **標題與裝飾**：赤陶棕 `#6D3B1E` / 酒紅磚 `#8B292A`
- **小 icon / hover**：摩卡棕 `#A68A6E` / 暖日黃 `#FAB44F`
- **深色元素與夜間版**：可可線條 `#6B4F39` / 深焙咖啡 `#21160F`

## 📱 響應式色彩應用

### 桌面版
- 使用完整色階漸層
- 大面積背景使用奶油燕麥
- 強調色使用暖日黃

### 平板版
- 簡化色階使用
- 保持主色焦糖淺棕
- 輔助色使用摩卡棕

### 手機版
- 最小化色彩使用
- 主色 + 白色背景
- 重點強調使用暖日黃

## 🔄 色彩轉換工具

### CSS 變數
```css
:root {
  --sova-primary: #B57E4F;
  --sova-secondary: #E8DFC9;
  --sova-accent: #FAB44F;
  --sova-text: #6B4F39;
  --sova-background: #FFF8EE;
}
```

### Tailwind 配置
```javascript
module.exports = {
  theme: {
    extend: {
      colors: {
        sova: {
          primary: '#B57E4F',
          secondary: '#E8DFC9',
          accent: '#FAB44F',
          text: '#6B4F39',
          background: '#FFF8EE',
        }
      }
    }
  }
}
```
